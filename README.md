## Tetris Game

A fully functional Tetris game built with HTML5, CSS3, and JavaScript!

### How to Play

1. Open `tetris.html` in your web browser
2. Use the following controls:
   - **← →** : Move piece left/right
   - **↓** : Soft drop (move piece down faster)
   - **↑** : Rotate piece
   - **Space** : Hard drop (instantly drop piece to bottom)
   - **P** : Pause/Resume game

### Features

- Classic Tetris gameplay with all 7 tetromino pieces
- Score tracking and level progression
- Next piece preview
- Smooth controls and rotation
- Increasing difficulty as you level up
- Responsive design with a modern UI
- Game over and restart functionality

### Scoring

- Clear 1 line: 100 points × level
- Clear 2 lines: 300 points × level
- Clear 3 lines: 500 points × level
- Clear 4 lines: 800 points × level
- Soft drop: 1 point per cell
- Hard drop: 2 points per cell

Level up every 10 lines cleared!
